# analisis_tic
En este repositorio verás el análisis de comunicaciones fijas y móviles con datos del Ministerio TIC de Colombia de los principales operadores de Telecomunicaciones del País. Desde validación de datos faltantes, manejo y manipulación, hasta diseño de gráficas para visualizar patrones y comportamientos y así tener conclusiones claras.
